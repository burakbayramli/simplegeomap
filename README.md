# simplemap

Given a center latitude, longitude and a zoom level simply plot all
country boundaries, rivers, lakes, elevation contour lines that fit on
the map. Each land feature can be disabled, enabled through its API
call.  The code works offline, all data needed to plot features on the
map are included along with the code for the program.

Datafiles, Preprocessing

Country Border Datafile comes from [here](https://thematicmapping.org/downloads/world_borders.php)

Lake data is from [NOAA](https://www.ngdc.noaa.gov/mgg/shorelines/data/gshhg/latest/)
